File Format Guidelines
CSV/TSV Files:
  • First row should contain column headers
  • Use comma (CSV) or tab (TSV) as delimiter
  • Avoid special characters in column names
  • Missing values can be empty or "NA"
Excel Files (XLSX):
  • Data should be in the first worksheet
  • First row should contain column headers
  • Avoid merged cells in data area
  • Remove formatting from data cells
Best Practices:
  • Keep file sizes reasonable (under 100MB for best performance)
  • Use descriptive column names
  • Ensure consistent data types within columns
  • Remove or document any preprocessing steps
